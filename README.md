# DriveHub

## 기획

## 🚗 1. 핵심 컨셉 정의

**사이트 이름(예시)**

- *DriveHub* / *로드메이트* / *GearUp* / *카밋(CarMeet)*

- 컨셉 문구 예시:

  > “브랜드를 넘어, 드라이브로 하나 되는 곳”
  >  “차쟁이들의 길 위의 커뮤니티”

**핵심 가치**

1. **브랜드 구분 없는 통합 커뮤니티** (BMW, 벤츠, 쉐보레 등 브랜드 상관없이)
2. **드라이브/와인딩 코스 공유**
3. **같이 탈 사람 찾기 (드라이브 파트너 매칭)**
4. **리얼 리뷰 / 차량 세팅 / 정비 노하우 공유**
5. **오프라인 모임·정모 문화 활성화**

------

## 🧭 2. 주요 페이지 구성

### ✅ [홈페이지]

- **메인 슬로건** + 대표 코스 사진 / 배너
- 오늘의 추천 코스 / 인기 게시물 / 이번 주 정모 안내
- 사용자 후기 카드 슬라이드
- “지금 근처에서 함께 달리는 사람” (위치 기반 추천)

------

### 🗺️ [드라이브 코스 맵]

- 전국 드라이브/와인딩 코스 지도 기반 표시 (구글맵, 카카오맵 API 활용)
- 필터 기능:
  - 🏞️ 경치 좋은 코스
  - 🌀 코너 많은 코스 (와인딩용)
  - ☕ 주변 카페/휴게 포인트
  - 🕓 주행 시간 / 거리 기준 필터
- 사용자 후기 + 차량 종류별 평점 표시
- “나만의 코스 등록하기” 기능 (코스 경로 + 후기 + 사진 업로드)

------

### 👥 [드라이브 메이트 찾기]

- 위치 기반, 차량 취향, 주행 스타일 기준 매칭
   예: “서울 출발, 토요일 오전, 와인딩 좋아함, 쉐보레/스포츠카 유저”
- 개인 프로필: 차량, 튜닝 상태, 좋아하는 코스, 주행 스타일
- 그룹 생성 기능 (정기 모임 / 브랜드 클럽 / 와인딩 크루)

------

### 🧑‍🔧 [정보 공유 & 커뮤니티]

- **게시판 분류 예시**
  - 자유게시판 (드라이브 후기 / 차량 자랑)
  - 정비/튜닝 노하우
  - 장비 추천 (타이어, 브레이크패드 등)
  - 중고거래 게시판
  - 브랜드별 소모임 (BMW / 쉐보레 / 현대 등)
- 게시판에 **사진 + 영상 업로드**, **해시태그 기능(#와인딩 #카마로)** 추가

------

### 📅 [이벤트 & 정모 캘린더]

- 지역별 모임 일정 달력 뷰
- 참가 신청 기능
- 주최자 프로필 및 장소 링크
- 과거 정모 후기 모음

------

### 🏁 [마이페이지]

- 나의 차량 프로필 (차종, 세팅, 사진)
- 내가 등록한 코스, 후기, 게시글
- 내 드라이브 메이트 리스트
- 참가한 이벤트 기록

------

## 💡 3. 차별화 포인트 아이디어

| 포인트                 | 설명                                                        |
| ---------------------- | ----------------------------------------------------------- |
| 🚘 **브랜드 통합성**    | 브랜드 구분 없이 “드라이브 경험 중심”으로 묶음              |
| 🗺️ **드라이브 코스 DB** | 전국 코스 + 유저 후기 데이터화                              |
| 🤝 **매칭 시스템**      | “같이 탈 사람 찾기” – 성격/스타일 기반 추천                 |
| 🔥 **감성 디자인**      | 차량 사진 중심의 다크 톤 UI (예: 매트 블랙 + 오렌지 악센트) |
| 📸 **SNS형 피드**       | 코스 후기, 사진, 짧은 영상 중심 피드 (인스타 감성)          |

------

## ⚙️ 4. 기술 구성 (간단하게)

- **프론트엔드**: Next.js or React + Tailwind CSS
- **백엔드**: Django / FastAPI / Node.js
- **DB**: PostgreSQL (지도/유저/게시글 관리)
- **지도 API**: Kakao Maps, Google Maps
- **인증**: 이메일 / 소셜 로그인 (카카오, 네이버, 구글)
- **AI 추천 (추후)**:
  - 유저의 드라이브 패턴 분석 → 코스 추천
  - 스타일 기반 메이트 추천

------

## 🎨 5. 디자인 키워드

- 다크 모드 중심
- 차량 사진이 돋보이게
- 지도 기반 시각화 강조
- “길 위의 자유로움” 느낌 (넓은 여백, 심플한 UI)

------

 → 홈, 코스 페이지, 커뮤니티 페이지 중심으로.



## 대략적 기능

### 🔧 지금 단계에서 하면 좋은 기획 정리 포인트

아래 세 가지를 정리해두면, 바로 와이어프레임이나 브랜드 콘셉트 디자인으로 넘어갈 수 있어요.

------

#### 1️⃣ **사이트의 방향성과 핵심 경험 (UX 컨셉)**

- 목적: 차쟁이들이 **브랜드 구분 없이 연결되는 드라이브 플랫폼**
- 사용자 행동 플로우:
  1. 로그인 → 2. 코스 구경 → 3. 같이 탈 사람 찾기 → 4. 후기 공유
- 감성 키워드: **자유로움 / 연결 / 도로 위의 설렘 / 나만의 길**

------

#### 2️⃣ **핵심 기능 구조도**

| 구분            | 기능                         | 비고                     |
| --------------- | ---------------------------- | ------------------------ |
| 홈              | 감성적 메인 배너 + 추천 코스 | 이미지 활용              |
| 드라이브 코스   | 지도 기반 검색 / 후기 / 등록 | Kakao or Google Maps API |
| 드라이브 메이트 | 위치·취향 기반 매칭          | 로그인 필요              |
| 커뮤니티        | 사진·후기·튜닝 정보 공유     | 브랜드별 태그            |
| 이벤트          | 정모·드라이브 일정 공유      | 달력 + 신청 기능         |
| 마이페이지      | 차량 프로필·활동 관리        | 개인화 추천 기반         |

------

#### 3️⃣ **브랜드 콘셉트**

| 항목      | 예시                                         |
| --------- | -------------------------------------------- |
| 톤 & 무드 | 다크 감성 + 따뜻한 주황빛 악센트 (#FF8C42)   |
| 키워드    | “길 위의 자유”, “함께 달리는 즐거움”         |
| 폰트      | San-serif 계열 (예: Pretendard, Inter)       |
| 로고 컨셉 | 도로 곡선 + 차량 실루엣 조합 (심플한 라인형) |
